Combat Patrol

Combat Patrol is a gamemode where your objective is randomly generated, most likely you will get a task to find and destroy something, the area will be guarded by enemy units.

It's best to play in a team of 4 people and more but you can play it alone.

Combat Patrol is probably the easiest gamemode to set up, all you need to do is to place the Combat Patrol Init. module.

There are few modules in this category, I'll list them below:

  • Combat Patrol Init. - Main module that creates the randomly generated missions.
  • Combat Patrol Location Add - Adds selectable position on the map.
  • Combat Patrol Location Remove - Removes selectable position on the map.
  • Combat Patrol Location Reposition - Repositiones selectable position on the map.
  • Combat Patrol Azimuth Blacklist - Bans area from having extraction point there.

Vanguard

Vanguard is a gamemode where your objective is to find, load and deliver special weapon back to your base. This gamemode is primarily focused on driving vehicles (mainly tanks).

It's best to play in a two teams of 8 and more but you can play it alone (without competition).

How to set up:

  1. Open your Systems: Modules: Gameplay Modes.
  2. Take the Vanguard: Objective Area module and place it on the ground.
  3. Then take 2 to 3 Vanguard: Starting Area modules and place them on the ground (Only one module for each faction).
  4. Place vehicles in the Starting Area modules and it's done.

Sector Control

Sector Control is a gamemode where 2-3 sided compete for complete dominance by capturing Sectors.

How to set up:

  1. Open your Systems: Modules: Multiplayer.
  2. Take the Sector module and place it on the ground.
  3. Now go to the Systems: Logic Entities: Location.
  4. From there take the Area logic entity and sync. it to the sector.
  5. And now go to the Systems: Logic Entities: Sides.
  6. There select the sided that would be fighting for that sector, place them and sync. them to the Sector.

Warlords

Warlords is a gamemode where two sides compete for complete dominance over the map.

Warlords gamemode is very similar to Sector Control, with few diffirences that you can buy units with money system.

How to set up:

  1. Open your Systems: Modules: Warlords.
  2. Grab all the essential modules and place them on the ground (Essential modules are: Warlords Init, two times Warlords Base, Warlords Sector).
  3. Warlords Base module must be synced. to the Warlords Sector modules.
  4. That's it.

Zeus

Zeus is a gamemode where player takes place as god, he can spawn / control units and many more.

It's very easy to set up, all you got to do is to place the Game Master module on the ground, all side modules must be synced. to the main one.

Zeus modules has a lot of types, I'll list the most important ones:

  • Game Master - Main module, sync. to the playable character.
  • Add Camera Area - Sets Area where Zeus can go.
  • Add Editing Area - Sets Area where Zeus can edit Assets.
  • Manage Resources - Sets how many resources Zeus would get.
  • Set Costs (Default) - Sets default cost of units.
  • Set Costs (Side) - Sets how much would every side cost.
  • Set Costs - Objects - Sets how much objects would cost.
  • Set Costs - Soldiers & Vehicles - Sets how much units would cost.
  • Zeus Lightning Bolt - Spawns a Lightning Bolt.
